🤖 Claude Code って何？

ひとことで言うと「日本語で指示するとパソコンが勝手に動くツール」

  • メールの下書き、議事録の整理、データ集計、画像作り、HPの作成、ブラウザ操作…
  • 全部、日本語で「○○して」と頼むだけで、PC上で自動的に作業してくれます。
  • ChatGPTとの違い: ChatGPTは「チャット画面の中だけ」。Claude Codeは「実際にPCを動かす」。

🎯 仕込み1: Plan Mode を使う（設計→実行）

大きな作業の前に必ず計画を立てさせるモード。コーディング前に手戻りが激減。

Claude Code起動中に Shift + Tab で「Plan Mode」に切り替え。

✅
Plan Modeを使うべき作業：HP制作・営業リスト処理・複雑なリファクタ・新機能追加
不要：1ファイル修正・メール下書き・単発リサーチ

📄 仕込み2: CLAUDE.md でプロジェクト学習させる

プロジェクトのルール・規約・前提を CLAUDE.md という名前でプロジェクトのトップに置くと、Claude Codeが毎セッション読みに行きます。

テンプレ：

# プロジェクト名 ## 言語・スタイル - 日本語で応答 - 簡潔・結論ファースト ## 頻出ミス記録（毎セッション読む） - ❌ ○○してはいけない → ✅ こう対応 - ミスのたびに追記して再発防止 ## ファイル構造 - src/ ─ メインコード - tests/ ─ テスト ## 品質基準 1. テスト必須 2. 型安全性 3. レスポンシブ対応

作り方の最速ルート: Claude Codeで /init と入力

/init

→ 自動でCLAUDE.mdの雛形ひながたを生成してくれます。あとはミスのたびに追記。

🔁 仕込み3: 検証ループで自動チェック

「コード変更したら必ずnpm testを実行する」みたいな自動チェックを仕込むと、AIが自分でミスに気づいて直してくれる。

方法: CLAUDE.md に「納品前チェックリスト」を書く → Claude Codeが完成報告前に自分で検証。

例（HP制作の場合）：

## 納品前チェックリスト - [ ] 誤字脱字なし - [ ] スマホ表示で崩れない - [ ] リンクが全て生きている - [ ] 個人情報が実データで正しい - [ ] テンプレ流用ではなくオリジナル
⚠️
3点セットの効果：
① Plan Mode = 手戻り激減
② CLAUDE.md = 同じミスを2回しない
③ 検証ループ = 納品物の品質が安定
合わせて使うと品質が2-3倍になります。

Q. AI触ったことないけど大丈夫？▼

大丈夫です。「日本語で指示するだけ」。プログラミング知識ゼロでもOK。研修当日は手取り足取り進めます。

Q. PCのスペックは？▼

普通の事務用PCで十分。Windows 10/11 または Mac（Intel/M1以降）。メモリ8GB以上推奨。ネット接続できればOK。

Q. 月額$20って高くない？▼

1日100円以下。1日30分の作業時短で元が取れる計算。ライティング・コーディング・調査・メール作成すべて加速。実質的にはコスト削減になります。

Q. 会社のPCでもインストールできる？▼

管理者権限が必要なケースあり。情シスに「Cursor + Node.js + Git のインストール許可」をもらってください。手動インストール（Step 3末尾）なら管理者権限なくても入る場合があります。

Q. データの安全性は？▼

Privacy Modeを有効にすればコード・データはAI学習に使われません。Cursor設定で Privacy Mode = Enable 必須（Step 2参照）。Anthropic（Claude）も商用APIではユーザーデータを学習に使わない方針。

Q. 既にChatGPT使ってる。Claude Codeとの違いは？▼

ChatGPT・Claudeのチャット版＝会話して提案をもらう。Claude Code＝実際にPC上でファイル編集・コマンド実行までやる（PCを操作するエージェント）。「指示すれば手も動く」のが最大の違い。

Q. CLAUDE.mdって必須？▼

必須ではないが絶対に作るべき。これがないと毎セッション同じ説明を繰り返す羽目に。/init で自動生成できるので、プロジェクト開始時に作る習慣を。

Q. 古いMacでも動く？▼

Intel Mac（2018以降）でも動きますが、M1/M2/M3 Macの方が圧倒的に快適。Cursor自体は古いMacでも動作します。
