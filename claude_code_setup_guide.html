🤖 Claude Code って何？

ひとことで言うと「日本語で指示するとパソコンが勝手に動くツール」

  • メールの下書き、議事録の整理、データ集計、画像作り、HPの作成、ブラウザ操作…
  • 全部、日本語で「○○して」と頼むだけで、PC上で自動的に作業してくれます。
  • ChatGPTとの違い: ChatGPTは「チャット画面の中だけ」。Claude Codeは「実際にPCを動かす」。

📚 用語ミニ辞典（途中で「これ何？」となる言葉）

セットアップ中によく出る用語をふつうの言葉で解説。先に読むとラクです。

用語かんたんな意味
ターミナル 文字でPCに命令する黒い画面（チャット風）。Cursorの中に表示される。Ctrl/Cmd+Jで開く。
コマンド ターミナルに打つ「PCへの命令文」。例: git --version = 「Gitのバージョン教えて」
パス（Path） PCの中でプログラムが置いてある場所のリスト。「パスが通ってない」＝PCがそのプログラムを見つけられない状態
環境変数 PCの設定みたいな値。「パス」も環境変数の一種。普段は気にしなくていい
Git（ギット） コードのバージョン管理ツール。「いつ・誰が・何を変えたか」を記録できる便利ツール
Node.js（ノードジェイエス） JavaScriptを動かす環境。Claude Codeの動作にも必要なので入れる
npm（エヌピーエム） Node.js用のアプリ管理ツール。npm install -g claude-code = 「Claude Codeを世界中どこでも使えるように入れて」
Homebrew（ホームブリュー） Mac用のアプリ・ツール管理ツール。brew install ○○ で何でも入る
winget（ウィンゲット） Windows用のアプリ・ツール管理ツール。Homebrewの Windows 版
シェル / Bash / PowerShell ターミナルの「中身の言語」。MacはBash/Zsh、WindowsはPowerShellが標準。普段は意識しなくてOK
sudo（スードゥ） Macで「管理者権限で実行」する命令。sudo ○○ でパスワードを聞かれる
API キー 外部サービス（Claude/OpenAI等）と接続するための「合鍵」。漏れたら課金事故になるので絶対に他人に見せない
パッケージ プログラムの「セット」。「Claude Codeパッケージをインストール」＝Claude Codeのプログラム一式を入れる
ディレクトリ / フォルダ 同じもの。MacやUnixでは「ディレクトリ」、Windowsでは「フォルダ」と呼ぶ習慣がある
リポジトリ（リポ） Gitで管理するプロジェクトのフォルダ。GitHubで公開するとURLになる
クローン（clone） GitHubからコードを自分のPCにダウンロードすること
プロンプト AIへの指示文。「メール書いて」みたいな。具体的なほど結果が良くなる
トークン AIが文字を処理する単位。日本語1文字≒1〜2トークン。Pro $20で月数百万トークン使える
ハルシネーション AIが「自信満々に間違える」こと。架空の数字や URL を作ることがある（要事実確認）
MCP（エムシーピー） AIと外部ツール（Chrome/Notion/Slack等）をつなぐ共通規格。Claude in Chromeもこれの一種

⚠️ Windows事前注意（必読・2点）

① wingetが使えるWindowsバージョンか確認

winget は Windows 11 または Windows 10 バージョン1709以降で動きます。古いWindows 10だと 'winget' is not recognized と出ます。

※ 自分のWindowsバージョン確認: Windows+R →「winver」と入力 → Enter。「バージョン1709」以上ならOK。

⚠️ Mac事前注意（必読・3点）

下の方法A/B/Cどれを選んでも、この3点は事前に把握しておくこと。知らないと途中で詰みます。

① Xcode Command Line Tools が自動DLされる（2-3GB / 10-20分）

Homebrewインストール時に「xcode-select --install」が自動で走ります。2-3GBのダウンロードが始まり、画面に「インストール」のダイアログが出ます。

  • 必ず「インストール」を選択（キャンセルするとHomebrewが動きません）
  • 10-20分かかるので、Wi-Fi安定環境＋電源接続で実施
  • 「これ何？」と止めない。Apple純正の開発者ツールで、Homebrewの動作に必須

② Apple Silicon (M1/M2/M3/M4) は PATH 追加が必須

Apple Silicon Mac は Homebrew が /opt/homebrew に入りますが、PATHを通さないと brew: command not found のままになります。Homebrewインストール後、必ず以下をターミナルで実行：

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile && source ~/.zprofile

※ Intel Mac の方はこの手順不要（自動でPATH通る）。M1以降の方だけ必須。
※ 自分のMacがどっちか分からない → 左上Apple🍎 → 「このMacについて」→ 「チップ」欄が「Apple M○」ならApple Silicon、「Intel Core ○○」ならIntel

※ 方法A（Cursorチャットに丸投げ）の場合、AIがこの設定も自動でやってくれることが多いですが、念のため確認

③ 会社支給Macで管理者権限がない場合

HomebrewもClaude Codeも管理者権限が必要です。会社支給で「sudoが拒否される」「インストールがブロックされる」場合は、研修1日前までに情シスに連絡してください。当日対応は間に合いません。

🔧 Mac固有のトラブル対応

症状対処法
「command not found: brew」 Homebrewインストール後にターミナル再起動。または以下を実行:

「App can't be opened」 Cursor.appを右クリック →「開く」→「開く」を再度クリック
M1/M2/M3 Macで「arch x86_64」エラー Rosetta 2のターミナルで実行する必要あり。arch -x86_64 brew ... 形式

Q. AI触ったことないけど大丈夫？▼

大丈夫です。「日本語で指示するだけ」。プログラミング知識ゼロでもOK。研修当日は手取り足取り進めます。

Q. PCのスペックは？▼

普通の事務用PCで十分。Windows 10/11 または Mac（Intel/M1以降）。メモリ8GB以上推奨。ネット接続できればOK。

Q. 月額$20って高くない？▼

1日100円以下。1日30分の作業時短で元が取れる計算。ライティング・コーディング・調査・メール作成すべて加速。実質的にはコスト削減になります。

Q. 会社のPCでもインストールできる？▼

管理者権限が必要なケースあり。情シスに「Cursor + Node.js + Git のインストール許可」をもらってください。手動インストール（Step 3末尾）なら管理者権限なくても入る場合があります。

Q. データの安全性は？▼

Privacy Modeを有効にすればコード・データはAI学習に使われません。Cursor設定で Privacy Mode = Enable 必須（Step 2参照）。Anthropic（Claude）も商用APIではユーザーデータを学習に使わない方針。

Q. 既にChatGPT使ってる。Claude Codeとの違いは？▼

ChatGPT・Claudeのチャット版＝会話して提案をもらう。Claude Code＝実際にPC上でファイル編集・コマンド実行までやる（PCを操作するエージェント）。「指示すれば手も動く」のが最大の違い。

Q. CLAUDE.mdって必須？▼

必須ではないが絶対に作るべき。これがないと毎セッション同じ説明を繰り返す羽目に。/init で自動生成できるので、プロジェクト開始時に作る習慣を。

Q. 古いMacでも動く？▼

Intel Mac（2018以降）でも動きますが、M1/M2/M3 Macの方が圧倒的に快適。Cursor自体は古いMacでも動作します。
